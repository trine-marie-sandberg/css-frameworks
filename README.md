## Getting Started

### Installing

1. Clone the repo:

```bash
git clone git@github.com:trine-marie-sandberg/css-frameworks.git
```
-or open with GitHub desctop. 

2. Install the dependencies (see the pacage.json file): bootstrap 5.2.2, live-server 1.2.2, and sass 1.55.0
